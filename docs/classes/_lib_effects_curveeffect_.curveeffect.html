modifyResize

  • modifyResize(widthTooSmall: boolean, heightTooSmall: boolean): void
    • Defined in lib/effects/CurveEffect.ts:409

    Changes the size of the object when called (when a corner guide is clicked and dragged).

    If any of width or height is too small, it sets them equal to 10 and the other equal to 10 divided or multiplied by the ratio of width/height to keep it the same.

    The work of changing the size is done by calling the helper method modifyResizeHelper.

    Parameters

    • widthTooSmall: boolean

      true if the width dimension is < 10

    • heightTooSmall: boolean

      true if the height dimension is < 10

    Returns void
